# ANALISI-ATTIVITA-DI-TRADING
Il progetto sfrutta Excel per monitorare ed estrarre insights significativi dalle attività di trading su varie piazze di stock exchange nel mondo

## DESCRIZIONE DEL PROGETTO
Questo progetto è stato sviluppato al termine del secondo modulo di Excel del Master in Data Analytics per mettere in pratica le competenza acquisite nell'ambito della **statistica descrittiva**.

### Foglio 1 - My trades
Il primo foglio offre la vista sull'intero datatbase, organizzato per:
- nome della Borsa
- informazioni geografiche
- quanità di azioni tradate per oeprazione
- prezzi in/out
- infromazioni temporali

### Foglio 2 - Stock Exchanges
La tabella mostra quante azioni sono scambiate per ogni area geoagrafica/Borsa.
Mette in luce anche eventuali tasse aggiuntive.

### Foglio 3 - Profit Insights
In questa sezione si analizzano i trades per valutarne l'efficienza. 

In particolare, si è cercato di capire se esiste una correlazione tra durata di detenzione di un pacchetto di azioni ed il profitto realizzato.

### Foglio 4 - Number of trades exchanged
La tabella contiene la distribuzione di frequenza marginale del numero di trades per Stato. Ciò è una base di partenza molto utile per estrapolare le probabilità di accadimento di tantissime tipologie di oeprazioni di trading.
